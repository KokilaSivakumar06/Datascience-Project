Project Overview

This project focuses on predicting maternal health risk levels using machine learning models based on physiological clinical data.

The system classifies pregnant women into:

Low Risk
Mid Risk
High Risk

The goal is to support early detection of high-risk pregnancies, enabling better clinical decision-making and improving maternal healthcare outcomes.# Datascience-Project

Objectives
Analyze clinical data related to maternal health
Build and evaluate multiple machine learning models
Identify key risk factors influencing maternal health
Improve prediction performance using feature engineering and class balancing
Develop an interpretable and reliable prediction system

Dataset
Source: UCI Machine Learning Repository
Total Records: 1014 instances
Type: Multi-class classification

Features:
Age
Systolic Blood Pressure
Diastolic Blood Pressure
Blood Sugar (BS)
Body Temperature
Hear Rate

Technologies & Tools
Python
Pandas, NumPy
Matplotlib, Seaborn
Scikit-learn
XGBoost
Imbalanced-learn (SMOTE)
SHAP (Explainable AI)

Models Used
Logistic Regression
Support Vector Machine (SVM)
Gradient Boosting
